Dependencias

El código depende de ciertas herramientas, concretamente:

  • CMake. Herramienta de generación o automatización de código multiplataforma.
  • OpenMP. API para facilitar la programación multihilo.
  • jsoncpp. Esta librería usa la licencia MIT. Se ha usado la versión amalgamada de jsoncpp, de modo que la estructura de directiorios queda de la siguiente manera:
    ext/
    ├── jsoncpp/
    │ ├── json/
    │ │ ├── json-forwards.h
    │ │ ├── json.h
    │ ├── jsoncpp.cpp
